# Demo

- Update `App.json` with your own bundle id that works on your device
- Run `npm install`
- Run `npm run ios`

## Reproduce the bug

- on the `Home Tab`, Click on `Go to Camera`
- Once the camera loads, click the `X` button on the top right
- Observe that the device freezes
